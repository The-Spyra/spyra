# Spyra - Portfolio

this is the portfolio site for spyra

## Prerequisites

- nodejs: v22
- editor with prettier support

## Local Setup

- Cone the repo

  ```bash
  git clone https://github.com/the-spyra/spyra && cd spyra
  ```

- Install Npm packages

  ```bash
  npm install
  ```

- Start the tailwind compiler on watch mode
  ```bash
  npm run dev
  ```
